# ATtiny13. Двухканальная регулируемая ШИМ 75КГц

![](schem.jpg)<br/>

### Возможности
- Кнопка «Пуск», которая не искрит из-за высоких стартовых токов при включении
- Регулировка мощности
- Плавный пуск 1с
